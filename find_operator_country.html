Main and sub bands
LTE/4G bands start with 'B' and 5G with 'n' and are called main bands. One OEM uses sub-bands such as 'B42C' to describe partial coverage of the main band. The left menu covers both main and sub-bands. The right main area covers main bands only, as sub-band and operator/country relationships are not publicly known.

Band and Operators

Can not find your band?
If an operator is migrating from LTE to 5G, they may appear with a B number but not yet an n number. Sub-bands (B42A, B78G, etc.) are not included here as the operator/region mapping is not publicly available. New bands such as n104 do not appear if not yet used by any tracked carrier.
